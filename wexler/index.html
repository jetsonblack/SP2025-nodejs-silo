MongoDB Commands for Lesson 16

Access Mongo Shell

/nobackup/jetblack/348/mongodb/mongodb/bin/mongo --port 12005

We use the above command to enter the Mongo shell.

List Databases

show dbs

We use this to see what databases exist on MongoDB.

Select Database

use confetti_cuisine

We use this to select the confetti_cuisine database for Lesson 16.

Show Subscribers Data

db.subscribers.find().pretty()

We use this to display subscriber data in a readable format.
